## Open Questions / Assumptions
- **SharedResource marker class location:** Assumed at project root `Tourist_Project_MVC/SharedResource.cs`. If the project uses a different namespace convention, adjust accordingly.
- **RTL scope:** Only EN and AR are required deliverables. ES/DE/ZH entries remain in the dropdown wired to the cookie but fall back to English strings.
- **Bootstrap RTL:** Because `styles.css` is compiled/baked, no CDN swap is possible. All RTL is handled via `[dir="rtl"]` attribute selectors in the new `rtl.css`.
- **Search icon glyph:** Kept as `bi-search` (standard, universally recognized). If a different glyph is preferred, it can be swapped in Task 1 without affecting wiring.
